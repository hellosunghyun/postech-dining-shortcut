학생식당의 오늘 조식, 점심, 저녁, 전체 식사를 단축어로 확인합니다.

포함 단축어

  • POSTECH 오늘 조식
  • POSTECH 오늘 점심
  • POSTECH 오늘 저녁
  • POSTECH 오늘 전체 식사

업데이트 방식

각 단축어는 실행 시 이 페이지의 version.json을 확인합니다. 새 버전이 있으면 다운로드 페이지를 자동으로 엽니다.

iOS Shortcuts는 사용자 확인 없이 단축어를 무음 교체할 수 없습니다. 설치 자체는 다운로드 후 직접 승인해야 합니다.

설치

  1. 최신 zip을 다운로드합니다.
  2. zip을 열고 원하는 .shortcut 파일을 실행합니다.
  3. Shortcuts 앱에서 설치를 승인합니다.
  4. 필요하면 iOS 자동화에서 Run Shortcut으로 연결합니다.

자동 빌드

GitHub Actions가 Cherri 소스를 빌드하고 signed .shortcut 파일을 Release zip으로 게시합니다.
