POSTECH 식단 단축어

원하는 단축어를 열어 설치하세요.

  • 포스텍 아침

    오늘 아침 메뉴를 확인합니다.

  • 포스텍 점심

    오늘 점심 메뉴를 확인합니다.

  • 포스텍 저녁

    오늘 저녁 메뉴를 확인합니다.

  • 포스텍 밥

    오늘 아침, 점심, 저녁 메뉴를 한 번에 확인합니다.

파란 단축어 이름을 누르면 파일을 내려받습니다. 내려받은 파일을 열어 단축어 앱에 추가하세요.

설치 후 “시리야, 포스텍 밥”처럼 단축어 이름을 말하면 실행할 수 있습니다.

각 항목은 하나의 단축어입니다. 설치 후 iOS 자동화에서 시간, 알림, 위치 같은 조건에 맞춰 실행할 수 있습니다.
